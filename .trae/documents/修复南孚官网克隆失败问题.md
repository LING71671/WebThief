## 问题诊断
南孚官网克隆失败的原因是 HTML 被错误保存为 JSON 格式，包含大量转义换行符 `\n` 而不是正常的 HTML 结构。

## 根本原因
`renderer.py` 中的 `_extract_dom_snapshot` 方法在提取 DOM 时，对于使用现代框架（Next.js/React）动态渲染的网站，无法正确获取最终渲染后的 HTML。

## 修复计划

### 1. 增强 DOM 提取可靠性 (renderer.py)
- 修改 `_extract_dom_snapshot` 方法
- 添加多种 DOM 提取策略（优先使用 `page.content()`，失败时回退到 `page.evaluate`）
- 增加内容验证，确保提取的是有效 HTML

### 2. 添加页面加载等待机制
- 在 `_navigate` 方法中增加对动态内容的检测
- 确保 JavaScript 完全执行后再提取 DOM

- 运行诊断脚本检查生成的 HTML 文件
- 确认修复后能够正确克隆南孚官网
