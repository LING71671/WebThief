## 测试计划

### 1. 运行克隆命令
使用以下命令测试对南孚官网的克隆：
```bash
python -m webthief https://www.nanfu.global/ -o nanfu_test_v4 --wait 5 -v
```

参数说明：
- `-o nanfu_test_v4`: 输出到新目录，避免覆盖旧结果
- `--wait 5`: 额外等待 5 秒确保页面完全加载
- `-v`: 启用详细日志，便于观察执行过程

### 2. 验证克隆结果
克隆完成后，检查生成的 HTML 文件：
- 文件大小是否合理（应该 > 1MB）
- 内容是否为有效的 HTML（不是 JSON）
- 是否包含正确的 HTML 标签结构

### 3. 对比修复前后
对比 nanfu_test_v3（修复前）和 nanfu_test_v4（修复后）的结果差异。

请确认后开始执行测试。
