# WebThief 路线图 (Roadmap)

> 文档维护更新（2026-02-24）：已清理仓库测试脚本与测试产物目录，本文档内容已同步调整。


## ✅ 已完成里程碑 (v3.0)
- 实时二维码拦截与桥接脚本注入
- React 组件卸载拦截与菜单状态冻结
- 高级功能 CLI 开关与配套文档完善

## 🚀 下一阶段 (v3.1)

1. 性能优化
   - 缩短菜单触发阶段耗时
   - 减少高级拦截带来的内存增量

2. 兼容性增强
   - 扩展 Vue/Angular 动态菜单拦截能力
   - 提升复杂 CORS 场景下二维码刷新稳定性

3. 可观测性提升
   - 增加二维码与菜单拦截诊断日志
   - 输出更清晰的失败原因与修复建议

## 📌 维护原则
- 优先保证克隆结果可运行、可验证
- 默认保持同 host 抓取边界和合规限制
- 新功能以可回退的配置开关方式上线
